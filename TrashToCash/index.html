Sambungkan ke Spreadsheet

Tempel URL Web App Google Apps Script ke variabel WEB_APP_URL di bagian atas file ini.

  1. Buat Google Spreadsheet baru.
  2. Buka Extensions → Apps Script, tempel isi file Code.gs.
  3. Klik Deploy → New deployment, tipe Web app, akses Anyone.
  4. Salin URL Web App → tempel ke WEB_APP_URL di index.html.
